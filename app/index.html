Micropayments for online media

Micropayments have for the long time been considered a holy grail for the Web. So close, yet impossible to achieve.

Ethereum payment channels technology allows us to build exactly what is craved for. Nice user experience included.

State channel compresses literally infinite amount of payments into few blockchain transactions. Less than a cent payment is finally feasible.
A consumer pays directly to a publisher. No middleman involved, no private data leaked. Safety of funds is secured by Ethereum.
Works in an ordinary web browser to eliminate friction. Supports ERC20 tokens for a thriving publisher's circled economy to thrive.

Cool Tech Inside

Vynos includes a secure identity vault, Web3 API to Ethereum network, and a micropayments API. Single line of JS code connects a web site to the blockchain. Vynos leverages Service Worker API and Same-Origin Policy enforced by modern browsers. It provides smooth identity management and micropayments experience on different web sites while preserving security.

This is a preview. Be the first to know when Vynos is generally available
